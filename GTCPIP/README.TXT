*---------------------------------------------------------------------*
* Ce Logiciel comporte des outils pour vous aider à administrer TCPIP *
*                                                                     *
*---------------------------------------------------------------------*

  Le premier outil  WRKTCPSTR
     Il permet de manager le démarrage des services TCP IP
     C'est le fichier QATOCSTART

  Le deuxième outil  WRKSVAUTE
     Il permet de manager les users de connexion distante
     Pour DRDA et DDM

  Le troisième outil LSTAPACHE
     Il permet de lister et de gérer le démarrage des instances
     Apache

*** Listes  des sources
